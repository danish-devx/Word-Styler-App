# Word Styler

![License](https://img.shields.io/badge/License-MIT-green)
![GitHub Repo](https://img.shields.io/badge/GitHub-Word%20Styler-blue)
![Stars](https://img.shields.io/github/stars/danish-devx/Text-Editor-App?style=social)

**Word Styler** is a lightweight, interactive text editor that allows styling individual words or entire text. It provides options for **bold, italic, underline, alignment, font size, text color, and background color** with a modern **dark-themed neumorphism interface**.

---

## Features

- **Bold / Italic / Underline** – Style individual words or the entire text.
- **Text Alignment** – Align text to left, center, or right.
- **Text & Background Color** – Choose custom colors.
- **Font Size** – Increase or decrease font size dynamically.
- **Copy to Clipboard** – Copy your styled text easily.
- **Reset** – Clear text and reset all styles.
- **Neumorphism Design** – Modern soft shadow interface.

## Usage

1. Type text in the editor.
2. Select a word or the entire text for styling.
3. Use the toolbar buttons to apply **bold, italic, underline, alignment, color, or font size**.
4. Click **Copy** to copy styled text.
5. Click **Reset** to clear content and styles.
